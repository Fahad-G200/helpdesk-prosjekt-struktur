1. REGISTRERING
   ├─ Bruker oppretter sak via web
   ├─ Kategori, prioritet og beskrivelse registreres
   └─ Automatisk bekreftelse (planlagt funksjon)

2. KLASSIFISERING
   ├─ Support vurderer prioritet
   ├─ Kategoriserer riktig (Feide, Wi-Fi, etc.)
   └─ Tildeler til rett team (ved behov)

3. UNDERSØKELSE OG DIAGNOSE
   ├─ Support følger kunnskapsbase
   ├─ Nivå 1 → Nivå 2 → Nivå 3
   └─ Dokumenterer funn i sakssystemet

4. LØSNING
   ├─ Implementer løsning
   ├─ Test at det fungerer
   └─ Dokumenter løsning

5. AVSLUTNING
   ├─ Bekreft med bruker
   ├─ Lukk sak i systemet
   └─ Oppdater kunnskapsbase hvis ny løsning
