# Brukerveiledning (kort)

## For elever/brukere
### 1) Start i Kunnskapsbase
Bruk Kunnskapsbase som nivå 1–2:
- Finn tema (Feide / Wi-Fi / Utskrift / Passord)
- Følg forslagene før du oppretter sak

### 2) Opprett sak (nivå 3)
Gå til “Mine saker” og fyll inn:
- Tittel (kort og presist)
- Beskrivelse (hva skjer, hva har du prøvd)
- Enhet/Programvare
- Prioritet

### 3) Legg ved skjermbilde/fil
Last opp PNG/JPG/PDF for å vise feilen.

---

## For support/admin
### 1) Se saker
Support ser alle saker og kan lukke dem.

### 2) Åpne vedlegg
Klikk vedlegg for å åpne i ny fane / laste ned.

### 3) Kunnskapsbase admin
Opprett og oppdater artikler slik at brukere kan løse mer selv (mindre tickets).

---

## Tips for god sak
Skriv alltid:
- tidspunkt
- feilmelding
- skjermbilde
- hva du har prøvd
