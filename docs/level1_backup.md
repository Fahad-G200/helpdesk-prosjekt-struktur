# Nivå 1 – Selvhjelp og standardløsninger

De fleste problemer kan løses her med enkle trinn.

## Feide-pålogging

**Symptom:** Innlogging feilet

**Løsninger:**
1. Sjekk brukernavn og passord
2. Velg riktig organisasjon
3. Sjekk internett
4. Prøv privat nettleservindu
5. Tøm nettleserdata

## Nettverk (WiFi)

**Symptom:** Ingen internett

**Løsninger:**
1. Sjekk WiFi er aktivert
2. Velg riktig nettverk
3. Start enhet på nytt
4. Glem og koble til på nytt

## Skriver

**Symptom:** Utskrift fungerer ikke

**Løsninger:**
1. Sjekk strøm
2. Sjekk papir og blekk
3. Velg riktig skriver
4. Omstart skriver

## Passord

**Symptom:** Glemt eller låst

**Løsninger:**
1. Bruk Glemt passord
2. Sjekk e-post
3. Vent 30 minutter

---

Hvis intet fungerer, gå til **nivå 2**.
